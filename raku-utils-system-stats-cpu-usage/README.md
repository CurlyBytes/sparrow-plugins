# raku-utils-system-stats-cpu-usage

Provides Statistics CPU Usage in percentage using System::Stats::CPUsage

# Istall

    s6 --install raku-utils-system-stats-cpu-usage

# Usage

    s6 --plg-run raku-utils-system-stats-cpu-usage
