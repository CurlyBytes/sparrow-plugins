# SYNOPSIS

Simple Perl Based password generator. Because `pwgen` is missing in Centos.

# INSTALL

    $ sparrow plg install pwgen

# USAGE

CLI:

    # generate password 16 symbols length
    $ sparrow plg run pwgen --param len=16
